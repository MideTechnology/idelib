1.4.2

Version 1.4.2 is a maintenance release, containing various improvements and minor bug fixes.

Changes in 1.4.2 include:

  • Improved timing accuracy in plots.
  • New "Tools" Menu, featuring GUI versions of the IDE file splitter and raw-to-MATLAB® command-line utilities.
  • New Slam Stick X™ configuration options, including USB attachment behavior and channel configuration.
  • New Slam Stick X™ firmware updating utility. This appears under the Device menu if the Show Advanced Options preference is checked. This feature requires firmware update files, which will be released in the near future.
  • Miscellaneous bug fixes and minor UI improvements.

1.4.1

Version 1.4.1 is a significant update, recommended for all users.

Changes in 1.4 include:

  • Multiple Axes per Plot
    • The most significant change in this version is the ability to plot multiple sources (axes) together in the main view. The majority of the changes in 1.4.x relate to this new functionality.
  • New View Menu Items
    • Create New Tab: Additional tabs can be created via this menu. Each tab is limited to a single measurement type, specified at the time of creation.
    • Display Channels: The display of individual channels of data can be turned on and off. Measurement types that are incompatible with the selected tab are disabled.
    • Show Legend: Toggle the floating legend display. The legend is described in more detail below.
    • 'Hollow' Envelope Drawing: With this mode turned on, the standard 'condensed' view (used when the number of points on screen exceeds the number of pixels) draws only the minimum and maximum, leaving the space between them empty. This is particularly useful when displaying multiple channels of data with differing amplitudes.
  • New Data Menu Item
    • Display Warning Ranges: Some types of sensor only produce reliable results in specific conditions. The 832M1 Accelerometer in the Slam Stick X™, for example, performs best at temperatures between -20 and 60°C; the default warning range displayed for accelerometer channels highlights periods during which the temperature is outside of this range. This replaces the single-purpose "Show Temperature Range Warnings" menu item, paving the way for additional sensors in future Midé data recorder products.
  • New Preferences
    • Initial Display Layout: This new option specifies the contents of the default tabs shown after loading a recording.
      • One Channel Per Tab: The old behavior.
      • One Tab Per Sensor: Channels of the same type, generated by the same sensor, are grouped together. For Slam Stick X™, this produces three tabs: one containing the three accelerometer axes, one containing pressure, and one containing temperature.
      • One Tab Per Type: Channels of the same type are grouped together, regardless of the sensor that recorded them. Note: For recordings made on current Slam Stick X™ hardware, this is identical to the One Tab Per Sensor option.
    • Legend Position: This specifies the corner of the screen on which the legend overlay is drawn. This applies specifically to the legend displayed in the main window, not rendered FFT/PSD/Spectrogram/Plot views.
    • Reset Plot Colors (button at bottom): User-edited plot colors are now saved; this option resets those colors to the defaults.
  • Legend Display: This overlay lists the colors and names of the channels currently displayed in a tab. Right-clicking items allows editing of the drawing order, plot colors, and the position of the legend.
  • Support for the latest Slam Stick X™ firmware
  • Various minor bug fixes and improvements.

1.3.0

Version 1.3.0 is a significant update, recommended for all users.

Changes include:

  • PSD (Power Spectral Density) calculation
    • Default setting uses the FFT results and plots results of g2/Hz versus frequency
    • A windowed calculation (using Welch's method) can also be selected that will plot dB/Hz versus frequency
      • This method provides better frequency information and will calculate faster
      • The smaller the window size the faster it will compute but the less "fine" the frequency information will be
  • Unit conversion
    • Display acceleration in g or m/s2
    • Display pressure in Pascals, PSI, or atm
    • Display pressure as altitude in meters or feet
    • Display temperature in °C, °F, or °K
  • Improved plotting
    • A feature under preferences, "Oversampling Multiplier", has been added which controls how many data points per pixel width of plot window is plotted
      • The default used to be 2, now it defaults to 25. Plotting will take about a second longer but many more data points will be shown. This setting is user selectable, the higher the number the more data that will be plotted; but the plot window will take longer to refresh.
  • Overall performance improvements.
  • New feedback feature under the Help menu
  • Various small UI fixes and improvements.

1.2.2

Version 1.2.2 contains several bug fixes and other minor improvements.

Changes include:

  • Temperature-based calibration (as applied to the accelerometer in Slam Stick X™ recordings) has been made substantially faster, improving performance in almost all operations.
  • Visual improvements to the Rendered Plot/FFT/Spectrogram views.
  • Intensive Spectrogram and FFT calculations are less likely to cause the application to appear frozen. FFT and Spectrogram calculation can also be interrupted.
  • Support for 2000 g Slam Stick X™ recorders.
  • Various small UI and performance improvements.

1.0.1

Version 1.0.1 primarily contains improved Slam Stick Classic™ support, but is recommended for all users.

Changes include:

  • Classic configuration 'limit number of recordings' now works correctly.
  • Classic recordings that were truncated prematurely will now load correctly.
  • Zero-length sessions will no longer appear in the 'select session' dialog that appears when opening a Classic recording.
  • Timestamps in Classic recordings are now handled correctly, and are displayed (as UTC) in the 'select session' dialog.
  • Various minor bug fixes.
