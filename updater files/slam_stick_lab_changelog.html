1.3.0

Version 1.3.0 is a significant update, recommended for all users.

Changes include:

  • PSD (Power Spectral Density) calculation
    • Default setting uses the FFT results and plots results of g2/Hz versus frequency
    • A windowed calculation (using Welch's method) can also be selected that will plot dB/Hz versus frequency
      • This method provides better frequency information and will calculate faster
      • The smaller the window size the faster it will compute but the less "fine" the frequency information will be
  • Unit conversion
    • Display acceleration in g or m/s2
    • Display pressure in Pascals, PSI, or atm
    • Display pressure as altitude in meters or feet
    • Display temperature in °C, °F, or °K
  • Improved plotting
    • A feature under preferences, "Oversampling Multiplier", has been added which controls how many data points per pixel width of plot window is plotted
      • The default used to be 2, now it defaults to 25. Plotting will take about a second longer but many more data points will be shown. This setting is user selectable, the higher the number the more data that will be plotted; but the plot window will take longer to refresh.
  • Overall performance improvements.
  • New feedback feature under the Help menu
  • Various small UI fixes and improvements.

1.2.2

Version 1.2.2 contains several bug fixes and other minor improvements.

Changes include:

  • Temperature-based calibration (as applied to the accelerometer in Slam Stick X™ recordings) has been made substantially faster, improving performance in almost all operations.
  • Visual improvements to the Rendered Plot/FFT/Spectrogram views.
  • Intensive Spectrogram and FFT calculations are less likely to cause the application to appear frozen. FFT and Spectrogram calculation can also be interrupted.
  • Support for 2000 g Slam Stick X™ recorders.
  • Various small UI and performance improvements.

1.2.1

Version 1.2.1 contains several bug fixes and other minor improvements.

Changes include:

  • FFT calculation has been reworked to improve accuracy and produce results in line with user expectations.

  • Improved accuracy when setting Slam Stick X™ clocks.
  • Various small UI and performance improvements.

1.2.0

Version 1.2.0 is a significant update, recommended for all users.

Changes include:

  • Export data directly to MATLAB© 5 .MAT format.
  • New Plot Render View: Graph multiple channels simultaneously, suitable for presentations.
  • FFT and Spectrogram View improvements, including:
    • Improved navigation and zooming of views.
    • Print directly from Slam Stick Lab.
  • The orientation of the Z axis in Slam Stick X™ recordings now matches the device's label.
  • Import and export sample count display shows the correct number of samples for all subchannels.
  • Improvements to the Slam Stick Classic™ configuration UI.
  • Various small bug fixes and performance improvements.

1.0.1

Version 1.0.1 primarily contains improved Slam Stick Classic™ support, but is recommended for all users.

Changes include:

  • Classic configuration 'limit number of recordings' now works correctly.
  • Classic recordings that were truncated prematurely will now load correctly.
  • Zero-length sessions will no longer appear in the 'select session' dialog that appears when opening a Classic recording.
  • Timestamps in Classic recordings are now handled correctly, and are displayed (as UTC) in the 'select session' dialog.
  • Various minor bug fixes.
